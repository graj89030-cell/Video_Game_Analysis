# 🎮 Video Game Sales & Engagement Analysis

## 📌 Project Overview
This project analyzes video game sales and user engagement data using Python and SQL.

## 🔧 Tools Used
- Python (Pandas, Matplotlib)
- MySQL
- Excel (Data Cleaning)

## 📊 Process
- Data cleaning performed in Excel and Python
- Exploratory Data Analysis (EDA) done using Python
- SQL used for querying and validating insights

## 🔗 Integration of SQL & Python
SQL analysis was used to validate and support insights obtained from Python-based exploratory data analysis.

## 📁 Files Included
- data_cleaning.ipynb → Python analysis
- games_cleaned.csv → Cleaned dataset
- vgsales_cleaned.csv → Sales dataset
- analysis_queries.sql → SQL queries

## 📈 Key Insights
- Game ratings and reviews show weak positive correlation
- Sales trends vary by region
- Popular genres dominate global sales
